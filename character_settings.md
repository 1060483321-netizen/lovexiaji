# 《恋与没意思》角色设定案 (进阶版)

本文档旨在完善《恋与没意思》中的核心角色设定，为后续剧情编写提供坚实的人格基础。

---

## 0. 我 / 主角 (Player)
**身份：** 大一新生
**关键词：** 观察者、微妙的吐槽、潜力股

| 维度 | 详细描述 |
| :--- | :--- |
| **性格细节** | 虽然是“没意思”主义者，但内心深处依然渴望着某种能够打破现状的火花。对周围的事物有着敏锐的洞察力（吐槽精准）。性格随和，容易被卷入各种麻烦中。 |
| **设定目的** | 作为玩家的载体，他需要足够的包容性，同时也要有自己的坚持。 |

---

## 1. 别凑 (Bie Cou) —— “极致理性的孤独者”
**关键词：** 冷酷、帅气、人气王、追求结果、隐藏狗奴

| 维度 | 详细描述 |
| :--- | :--- |
| **语言风格** | 简短、有力、富有逻辑。很少使用感叹号，经常用“结论是……”开头。 |
| **台词示例** | “我不需要虚伪的赞美。如果你做不到，那就退到一边，不要浪费大家的时间。……等等，那只流浪狗还没吃午饭。” |
| **剧情钩子** | 别凑完美的履历背后，是家族的高压期待。她那种拒人于千里之外的冷酷，其实是在保护自己不被看穿脆弱。 |

---

## 2. 种芒 (Zhong Mang) —— “糖衣下的深渊”
**关键词：** 娃娃脸、腹黑、情报贩子、掌控欲、内心荒芜

| 维度 | 详细描述 |
| :--- | :--- |
| **语言风格** | 甜腻、常用敬语但语气轻佻。喜欢用疑问句引导对方说出不该说的话。 |
| **台词示例** | “哎呀，林秋同学，你刚才和学姐聊得很开心呢~ 要是这份秘密被别的人知道，会发生什么有趣的事情呢？我很期待哦~” |
| **剧情钩子** | 种芒并不信任人类，对他而言，所有关系都是“交易”。让这个腹黑的角色真心微笑，是攻略他的核心难点。 |

---

## 3. 肥皂 (Fei Zao) —— “地平线的追逐者”
**关键词：** 温和、可靠、户外运动、自由、活在当下

| 维度 | 详细描述 |
| :--- | :--- |
| **语言风格** | 爽朗、直接、带有一点点大姐姐的调侃。笑声非常有感染力。 |
| **台词示例** | “别总是盯着天花板看，出门骑一会儿，那片海的角度可是每天都在变的。来吧，这次我载你。” |
| **剧情钩子** | 虽然看起来最可靠，但肥皂其实对未来感到迷茫。她这种“活在当下”的姿态，实际上是对即将到来的社会生活的某种逃避。 |

---

## 4. 止水 (Zhi Shui) —— “暴风雨前的宁静”
**关键词：** 老实人、透明感、极度坦率、深不可测

| 维度 | 详细描述 |
| :--- | :--- |
| **语言风格** | 语速平缓，一板一眼。有时会坦率到令人尴尬的地步。 |
| **台词示例** | “我不明白为什么大家要撒谎。如果你不喜欢这顿饭，直接说‘很难吃’不是更有效率吗？就像我现在的感觉一样。” |
| **剧情钩子** | 被称为“老实人”是因为她拒绝参与社交潜规则。这种纯粹的真实，在复杂的校园关系中其实是最具杀伤力的武器。 |

---

## 5. 小葵 (Xiao Kui) —— “孤独的电波少女”
**关键词：** 黑长直、电波系、音游大触、宅力、笨拙

| 维度 | 详细描述 |
| :--- | :--- |
| **语言风格** | 引用大量ACG黑话，有时会突然陷入沉默（在脑内模拟Combo）。 |
| **台词示例** | “这个场景的BGM太悲伤了，不符合目前的攻略进度……啊，我刚才是不是把内心的OS说出来了？总之，请无视吧，愚蠢的二次元……” |
| **剧情钩子** | 美丽的外表让她在现实中受到排挤。她躲进游戏世界是因为只有在那里，规则是公平的。她最需要的不是被当成女神，而是被当成一个普通的女孩子对待。 |

---

## 6. 杨文明 (Yang Wenming) —— “被磨灭的微光”
**关键词：** 颓废、社畜、毒舌、 mentor (导师)、意外的浪漫

| 维度 | 详细描述 |
| :--- | :--- |
| **语言风格** | 疲惫、重度社交恐惧症的表现、冷幽默。常伴随着叹气和喝咖啡的声音。 |
| **台词示例** | “现在的年轻人真是活力过剩啊……等你们工作三年，看什么都是灰色的。不过，偶尔看看你们这类没有意义的烦恼，倒也能让我想起自己还没彻底变成石头。” |
| **剧情钩子** | 作为唯一的成年人，他是主角在踏入社会前的镜子。他的存在是为了告诉主角，虽然生活很“没意思”，但即使是这种没意思的生活中，也藏着需要守护的东西。 |

---

## 7. 神秘墨镜女生 (暂用代号 SY) —— “危险感像玩笑，也像警告”
**关键词：** 墨镜、黑衣、压迫感、冷香、真假难辨、校外遭遇

| 维度 | 详细描述 |
| :--- | :--- |
| **当前身份** | 现阶段剧情里姓名未公开，仅以 `sy` 立绘和街头初遇事件登场。她给人的第一印象像“最好别惹”的校外危险人物，但是否真的是混社会的人，还保留悬念。 |
| **语言风格** | 句子短，停顿多，几乎不提高音量，却能把压迫感稳稳压到对方身上。比起真正暴怒，更像冷冷地看着对方自己慌。 |
| **剧情钩子** | 她在主角第一次校外采购时出场，以“撞了人，道歉就完了？赔钱”制造强烈记忆点。关键魅力在于暧昧性: 她到底是在认真勒索、随手试探，还是带着某种只有她自己知道的玩笑心态，目前都不能说死。 |

## 角色关系关系 (Relationship Matrix)

1. **别凑 & 种芒 (权力的角逐)**：相互利用又相互提防。别凑欣赏种芒的效率，种芒忌惮别凑的直觉。
2. **小葵 & 止水 (同频不同步)**：两个在日常社交中“掉线”的人。经常出现两人对话各说各的，却能完美衔接的奇观。
3. **肥皂 & 杨文明 (理想与现实)**：一个在拼命追逐自由，一个在拼命忍受沉重。他们对彼此的生活方式都有一种复杂的向往。
4. **主角 & 全员**：主角是那块让所有人的人生齿轮（无论生锈与否）重新转动的关键零件。
