# 场景补充建议

基于 `game/story_preview.rpy` 与 `game/assets_map.rpy` 当前版本整理。

## 先说结论

当前场景资源最大的短板不是数量太少，而是“很多名字已经很具体的场景，还在拿通用背景硬顶”。

最明显的几类问题：

- 很多剧情专用场景仍在用 `campus_bg.png`
- 很多私密室内场景仍在用 `dorm_bg.png`
- 很多夜间约会 / 酒局 / 送人回家场景仍在用 `street_night.png`
- 很多后台 / 演出 / 漫展场景仍在用 `school_stage.png`
- 很多办公室权力戏仍在用 `student_office.png`
- 很多聚餐场景仍在用 `cafeteria_bg.png`

所以补场景时，最值钱的是补“专用剧情空间”，不是继续补泛校园。

## 第一优先级：最该补的新场景

### 1. `basement_bandroom`

- 当前映射：`campus_bg.png`
- 章节参考：第二十五章，3257 行附近
- 问题：
  - 剧情写的是地下排练室、阴冷、潮湿、重金属感。
  - 现在却还是普通校园背景，气氛完全不对。
- 建议：
  - 做成地下室排练房。
  - 要有音箱、线材、贴纸、旧墙、器材架。

### 2. `instrument_shop_day`

- 当前映射：`campus_bg.png`
- 章节参考：第二十五章，3270 行附近
- 问题：
  - 这是别凑和小葵关系线里很可爱的关键场景。
  - 乐器店如果没有专属背景，记忆点会掉很多。
- 建议：
  - 墙上挂吉他、效果器柜、试琴凳、暖光。

### 3. `hotel_room_night` / `hotel_room_morning`

- 当前映射：`dorm_bg.png`
- 章节参考：第十九章，2744 行附近；第二天清晨 2764 行附近
- 问题：
  - 这是一段关系发生实质变化的核心节点。
  - 宿舍背景顶酒店房会非常出戏。
- 建议：
  - 夜晚版和清晨版至少做同一空间的两种光线。
  - 要有大床、床头灯、窗帘、酒店感。

### 4. `haunted_school`

- 当前映射：`campus_bg.png`
- 章节参考：第十四章，2163 行附近
- 问题：
  - 剧情明确写了废校风、血字、鬼屋布景。
  - 现在用普通校园完全撑不住。
- 建议：
  - 破旧教室、蓝暗光、血字、临时鬼屋道具。

### 5. `rollercoaster_ride`

- 当前映射：`campus_bg.png`
- 章节参考：第十四章，2113 行附近
- 问题：
  - 这是关系升温的重要约会节点。
  - 过山车如果还是校园背景，会非常跳戏。
- 建议：
  - 可以不画很复杂，但至少要有轨道、俯冲视角、速度感。

### 6. `chairman_office_day`

- 当前映射：`student_office.png`
- 章节参考：第二十八章，3447 行附近
- 问题：
  - 会长办公室是肥皂 x 杨文明关系线的高张力空间。
  - 普通学生会办公室不够。
- 建议：
  - 做成更高级、更有压迫感的独立办公室。
  - 会客沙发、书柜、落地窗、办公桌分区都很重要。

### 7. `family_dinner_night`

- 当前映射：`dorm_bg.png`
- 章节参考：第三十三章，3825 行附近
- 问题：
  - 小葵家族宴会的“灯火辉煌、礼裙、寒暄”氛围很强。
  - 用宿舍背景会直接毁掉这段戏。
- 建议：
  - 高级家宴厅、吊灯、长桌、玻璃反光、偏金色调。

### 8. `band_venue_backstage`

- 当前映射：`school_stage.png`
- 章节参考：第二十六章，3334 行附近
- 问题：
  - 商演后台和校园舞台后台气质不一样。
  - 这里还承载“别凑第一次看到成人世界的脏与恶”。
- 建议：
  - 更拥挤、更商业、更杂乱。
  - 化妆镜、线缆箱、啤酒箱、贴纸墙都可以有。

## 第二优先级：很值得补，但可以排在上面之后

### 9. `external_affairs_office`

- 当前映射：`student_office.png`
- 章节参考：第四十六章、第三十章、第四十五章附近
- 问题：
  - 外联 / 经费 / 权力施压的办公室，和学生会日常办公室不应完全同景。
- 建议：
  - 更像行政办公室，文件柜、预算单、压抑白光。

### 10. `office_window_night`

- 当前映射：`student_office.png`
- 章节参考：第十章、第四十五章、第五十章等
- 问题：
  - 这是“小葵母亲办公室 / 成人世界压力”的代表空间。
  - 应该和校园办公室明显区分。
- 建议：
  - 夜景落地窗、城市灯光、冷调、高处俯视感。

### 11. `internet_cafe_day`

- 当前映射：`campus_bg.png`
- 章节参考：第二十五章，3283 行附近
- 问题：
  - 网吧面基是 SY 和余晖这条线最有识别度的场面。
- 建议：
  - 电脑排位、霓虹灯、隔断、饮料杯。

### 12. `comic_con_day`

- 当前映射：`school_stage.png`
- 章节参考：第二十一章，2861 行、2946 行
- 问题：
  - 漫展和普通舞台不是一回事。
- 建议：
  - 展板、摊位、人流、周边墙、摄影区。

### 13. `bbq_night`

- 当前映射：`street_night.png`
- 章节参考：第二十八章，3435 行
- 问题：
  - 肥皂和止水单独烧烤，是关系线比较有味道的私下场景。
- 建议：
  - 烧烤摊、桌面、夜灯、炭火、烟气。

### 14. `bbq_soju_night`

- 当前映射：`street_night.png`
- 章节参考：第十七章，2528 行；第八章，1505 行
- 问题：
  - 酒局、情绪崩溃、靠怀里哭，这种戏很需要桌面和空间感。
- 建议：
  - 韩式烧酒小店或夜宵摊，桌上酒瓶、烤盘、纸巾。

### 15. `taxi_night`

- 当前映射：`street_night.png`
- 章节参考：第五章、第三十四章
- 问题：
  - 阿表送醉酒种芒回校那段很有氛围。
  - 出租车内部景比街景更有情绪。
- 建议：
  - 后座视角、窗外雨痕、路灯反光。

### 16. `pharmacy_night`

- 当前映射：`street_night.png`
- 章节参考：第五章，1130 行
- 问题：
  - 深夜买药是典型关系推进小场景。
- 建议：
  - 便利药店门头、白灯、收银台或门口橱窗。

## 第三优先级：有条件再补

### 17. `beef_intestine_room_night`

## 反而没那么急着补的

这些虽然也有复用，但暂时还扛得住：

- `basketball_court_match`
- `basketball_court_day`
- `arena_stands_day`
- `infirmary_day`
- `dance_room_day / night`
- `zhongmang_room_night`
- `xiaokui_room_day / night`
- `school_gate_day`

原因：

- 这些场景类型和当前代用图至少大方向一致。
- 先补它们的收益，不如先补那些“名字和画面完全不符”的场景。

## 一种更省资源的做法

如果你不想一次补太多，可以先做 4 张“通用升级图”：

1. 乐队地下排练室
2. 高级办公室夜景
3. 酒店房间夜 / 晨
4. 夜间餐酒小店

这 4 张就能立刻覆盖掉一大批目前最出戏的章节。
