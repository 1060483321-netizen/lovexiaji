# 服装与双人演出建议

基于 `game/story_preview.rpy` 当前 58 章预览版本整理。

## 先说结论

- 低戏份角色不建议单独做立绘。
- 这一版最该补的不是“更多人头”，而是主角组的换装、关系线双人 CG、以及少数高记忆点事件图。
- 最值得优先投入的三条线：
  - `SY x 种芒`
  - `别凑 x 小葵`
  - `肥皂 x 杨文明`
- 次优先但很有风味的一条线：
  - `肥皂 x 止水`

## 不建议单独投入立绘的人

- `shortgirl`：第一章功能性路人。
- `bandleader`：有作用，但更适合做半身剪影、背影或场景内人物，不必正式立绘。
- `mother`：前期功能明确，但戏份太少，继续保留文字或剪影更省资源。
- `yuhui`：戏份有记忆点，但暂时不值得优先做整套立绘。

## 主要人物该补什么服装

### SY

建议至少做 4 套：

- 校园常服。
- 篮球训练服或运动服。
- 黑色西装。
  - 对应第二章督场，`game/story_preview.rpy` 450-451 行。
- 冬季或约会私服。
  - 主题乐园、平安夜、后期街头和见面戏都能复用。

如果预算够，再加 2 个特殊版本：

- 受伤后或吹头发场景用的居家版本。
- 酒后或晨起的轻微凌乱版本。

### 种芒

建议至少做 5 套：

- 日常舞社私服。
- 舞蹈练习服。
- 迎新晚会 / MOSA 舞台演出服。
  - 第一章和第二章都用得上，198-238 行、450-489 行。
- 情绪低谷期的夜晚外出私服。
  - 第三十三章开始频繁出入酒吧，3829-3837 行。
- 结尾前后的室内私服。
  - 她和 SY 的大量核心戏都发生在房间里。

她是整部里最值得做“发型和妆面区分”的人：

- 舞台版要明显更亮、更攻击性。
- 室内版要更柔软、更疲惫。

### 别凑

建议至少做 5 套：

- 篮球线早期运动服。
- 校园常服。
- 乐队排练服。
  - 第二十五章起正式进入乐队线，3261-3275 行。
- 商演 / 舞台服。
  - 第四十一章 Ark 演出，4096-4110 行。
- 情绪崩溃期的宽松居家服或卫衣。
  - 第十七章和第五十章都能复用，2526-2558 行、4388-4397 行。

### 小葵

建议至少做 5 套：

- 校园常服。
- 舞蹈练习服。
- 舞社演出服 / 啦啦队服。
  - 第四十二章三对三比赛当天，4134 行。
- 家族宴会礼裙。
  - 第三十二章和第三十三章，3810-3813 行、3829-3833 行。
- 陪伴别凑时期的居家私服。
  - 第五十章后期同住线，4388-4397 行。

### 肥皂

建议至少做 5 套：

- 篮球队长球衣。
- 日常运动便服。
- 漫展 / 机车出行私服。
  - 第二十一章，2918-2950 行。
- 办公室线的稍正式一点的便服。
  - 第二十八章，3452-3473 行。
- 受伤后版本。
  - 最少要有护膝或轮椅版本，第五十八章 4644-4645 行。

### 止水

建议至少做 3 套：

- 舞社常服或练习服。
- 机车外出装。
  - 第二十一章，2918-2944 行。
- 漫展外出装。

她不用做很多，但“黑色机车 + 冷脸”这一套必须有。

### 杨文明

建议至少做 4 套：

- 学生会正式套装。
- 日常会长装。
- 居家版或私下版。
  - 她和肥皂的关系戏需要一套更卸下防备的版本。
- 情绪压迫感更强的办公室版本。

她的问题不是服装数量不够，而是“所有戏都不能只是一张 formal 脸”。

## 最值得做成双人立绘或事件 CG 的场面

以下按制作价值排序。

### 第一梯队：必做

#### 1. `SY x 种芒` 舞台后台饭卡递还

- 章节：第二章
- 参考：450-489 行
- 画面价值：
  - SY 黑西装站在暗处。
  - 种芒刚下台，仍在演出状态里。
  - 一明一暗，第一次真正近距离对视。
- 这是两人关系线第一个非常适合做事件 CG 的瞬间。

#### 2. `SY x 种芒` 主题乐园过山车压手 / 喂冰沙

- 章节：第十四章
- 参考：2101-2140 行
- 建议拆成 1 张主 CG 或 2 张小 CG：
  - 过山车上，种芒把手压到 SY 手背上。
  - 长椅上喂冰沙。
- 这是典型的“关系升温而且画面好看”的场面。

#### 3. `SY x 种芒` 酒店夜与次日晨

- 章节：第十九章
- 参考：2736-2764 行
- 建议：
  - 夜间做克制版，不必直白。
  - 重点画抓手、靠近、床边光影、第二天晨光后的沉默。
- 这是关系实质变化节点，必须有画面承托。

#### 4. `SY x 种芒` 吹头发

- 章节：第四十三章
- 参考：4167-4171 行
- 这张非常适合做安静的室内半身 CG。
- 气质要软，和前面的舞台感形成反差。

#### 5. `SY x 种芒` 结尾流泪拥抱

- 章节：第五十八章
- 参考：4659-4671 行
- 这是整条线最该有的一张收束图。
- 不一定复杂，但情绪必须打中。

### 第二梯队：非常值得做

#### 6. `别凑 x 小葵` 黑房间草莓告白

- 章节：第十七章
- 参考：2526-2558 行
- 画面点：
  - 黑暗房间。
  - 小葵带着奶油草莓找上门。
  - 别凑情绪崩塌，小葵试图接住她。
- 很适合做一张低饱和、很近距离的情绪 CG。

#### 7. `别凑 x 小葵` 乐器店买琴

- 章节：第二十五章
- 参考：3273-3275 行
- 这张不一定是重情绪，但非常适合做“关系可爱期”的双人图。
- 日落配色 Fender 和天蓝色拨片都很有记忆点。

#### 8. `别凑 x 小葵` 创伤后陪睡

- 章节：第五十章
- 参考：4388-4397 行
- 重点不是暧昧，是“守着你睡着”那种安全感。
- 适合做夜灯、床边、安抚型构图。

#### 9. `肥皂 x 杨文明` 办公室越界

- 章节：第二十八章
- 参考：3452-3473 行
- 这是整部里最适合做高张力双人图的场面之一。
- 关键词：
  - 会长办公室
  - 逼近
  - 会客沙发
  - 外部打断
- 非常适合单张高压 CG。

#### 10. `肥皂 x 止水` 机车与抱腰

- 章节：第二十一章
- 参考：2918-2944 行
- 这张不一定要做成全屏 CG，也可以做特殊双人立绘：
  - 止水在前。
  - 肥皂抱腰。
  - 机车、风、路边城市线条。

### 第三梯队：有预算就上

## 最推荐的制作顺序

如果你想最省钱又最出效果，建议按这个顺序做：

1. `SY`、`种芒` 常驻服装包。
2. `别凑`、`小葵` 常驻服装包。
3. `肥皂`、`杨文明`、`止水` 的关系线专用服装。
4. 先做 5 张第一梯队 CG。
5. 再做 3 张第二梯队 CG。
6. 最后才补零散场景奖励图和边缘角色。
